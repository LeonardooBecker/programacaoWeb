Plataforma de coleta e painel de visualização dos dados extraídos

Plataforma de coleta

Para realizar a captura das imagens dos contudores e dos dados de GPS para cada viagem efetuada pelos participantes da pesquisa, foi utilizado um conjunto de duas camêras veículares que possuem a geolocalização integrada ao seu sistema.

As camêras foram dispostas internamente ao veículo. A primeira delas aponta para o interior do automóvel, visando coletar as ações como mexer no telefone celular, interagir com painel digital do carro, entre outros. Já a segunda é responsável por registrar as imagens frontais do veículo, ou seja, tudo que acontece externamente interferindo em cada ação tomada pelo condutor.

Primeiramente foi desenvolvido utilizando as linguagens C, Shell e Python, um software responsável
